Anaconda Environment
Open anaconda prompt app and on the command line type
conda activate quantra.py

Set-up
The onetime set-up, clone of the quantra enviroment can also be peformed by typing the following command into the anaconda prompt
conda env create -f https://quantra.quantinsti.com/downloads/machinelearning/environment.yml

Python Version Used
python 3.9.5

Environment List
conda info --envs
